Sobre esta página

Esta é uma página de exemplo para testar a biblioteca acessibilidade-br. Use o widget flutuante no canto da tela para experimentar todas as funcionalidades de acessibilidade.

Visite o site oficial do W3C WAI para entender melhor as diretrizes de acessibilidade.

Conteúdo para testar Texto para Fala

Quando você ativar a opção Texto para Fala, qualquer parágrafo ou bloco de texto que você clicar será lido em voz alta pelo navegador. Experimente clicar em diferentes elementos desta página enquanto a função estiver ativa.

A síntese de voz utiliza a Web Speech API nativa do navegador. A qualidade da voz e os idiomas disponíveis dependem do sistema operacional e do navegador que você está usando.

Conteúdo para testar tamanhos e espaçamentos

Use os botões Aumentar Texto e Diminuir Texto para ajustar o tamanho da fonte em toda a página. Os botões de Espaçamento ampliam ou reduzem a distância entre letras, palavras e linhas, melhorando muito a legibilidade para pessoas com dislexia ou baixa visão.

Inverter Cores: útil para usuários com sensibilidade à luz.
Tons de Cinza: remove cores que podem distrair.
Sublinhar Links: destaca todos os links da página.
Fonte para Dislexia: aplica fonte mais legível.
Cursor Grande: facilita ver o ponteiro do mouse.
Guia de Leitura: destaca a linha que você está lendo.
